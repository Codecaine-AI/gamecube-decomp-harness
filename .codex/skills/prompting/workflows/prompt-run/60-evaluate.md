# Evaluate

Evaluate every artifact before final delivery.

## Steps

1. Read `../../30-generation/06-evaluation-protocol.md`.
2. Score all five dimensions.
3. Record top issue and specific fix for each dimension.
4. Apply fixes when the average is below the required bar.
5. Re-evaluate after fixes.
6. Save the final evaluation in `steps/06-evaluation.md`.

## Formatter Adjustment

For input formatters, replace Reasoning Architecture with Contract Adherence:

- Does the rendered message satisfy the system prompt's input contract in field names, shape, order, and inclusion rules?

## Output Table

Include the final scores table in both `steps/06-evaluation.md` and `outputs/final-report.md`.
