Method Notes

The stats JSON was rebuilt with scripts/analyze-pi-agent-tools.py while applying an in-memory transcript-path fallback from the old repo path to this checkout path for the comparison run. Branch-delta counts come from projects/melee/checkout/build/GALE01/report_changes_production.json, so they describe the current WIP checkout versus production rather than one lease per worker.
