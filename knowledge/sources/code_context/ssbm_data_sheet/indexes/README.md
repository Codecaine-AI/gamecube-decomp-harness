# SSBM Data Sheet Indexes

Source-local exports:

- `codebase_facts.jsonl` is generated by
  `commands/build_codebase_facts.py`. It contains current checkout function
  facts, symbols, source references, and local curator lessons.
- `codebase_facts.meta.json` records the checkout inputs used to build
  `codebase_facts.jsonl` so status and `--check` can report staleness.
- `cells.jsonl` is generated during `kg-rebuild-graph` from both legacy
  workbook CSVs under `data/csv` and generated CSVs under `data/generated`.

Worker sessions should query the API scripts instead of parsing the workbook or
generated CSVs directly.
